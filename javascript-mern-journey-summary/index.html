Exploring the building blocks of web interaction and data fetching.

Important Topics & Concepts:

  • Asynchronous JavaScript with fetch(): Understanding how to make non-blocking requests to external APIs to retrieve data. 

fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => {
    console.log(data);
  })
  .catch(error => {
    console.error('Error fetching data:', error);
  });

Day 5: Async/Await Implementation

Simplifying asynchronous code with modern syntax.

Features:

  • Random User Dashboard: A dashboard-style application.
  • Async data loading: Fetching data asynchronously using `async`/`await`.

async function fetchData() {
  try {
    const response = await fetch('https://api.example.com/data');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('Error fetching data:', error);
  }
}

fetchData();
  • Loading states and animations: Providing visual feedback while data is loading.
  • Error handling with try/catch: Implementing robust error handling in asynchronous functions.

Technical Highlights:

Day 9.10: Timer

Implementing time-based functionality and task management.

Important Topics & Concepts:

  • setInterval Implementation: Using `setInterval` for recurring actions.

let count = 0;
const intervalId = setInterval(() => {
  count++;
  console.log('Count:', count);
  if (count >= 5) {
    clearInterval(intervalId);
    console.log('Timer stopped.');
  }
}, 1000);
  • Timer State Management: Tracking the timer's status (running, paused, stopped).
  • Task Management: Adding and displaying completed tasks.
  • Progress Tracking: Visualizing the timer's progress (e.g., with a progress bar).
  • Mode Switching: Handling different timer modes (countdown, stopwatch, pomodoro).
